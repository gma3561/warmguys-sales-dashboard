🔒 보안 Google Sheets 연동

🛡️ 보안 연동 방식

이 방식은 Google Sheets를 공개하지 않고도 안전하게 데이터를 가져올 수 있습니다. Google Apps Script 웹 앱을 통해 인증된 요청만 처리합니다.

📋 설정 단계

  1. Google Apps Script 설정:
    • Google Apps Script에서 새 프로젝트 생성
    • google-apps-script/Code.gs 파일의 내용을 복사하여 붙여넣기
    • 스프레드시트 ID와 시트 이름을 실제 값으로 수정
    • API 키를 안전한 값으로 변경
  2. 웹 앱 배포:
    • Apps Script에서 "배포" → "새 배포" 선택
    • 유형: "웹 앱" 선택
    • 실행 권한: "나" 선택
    • 액세스 권한: "모든 사용자" 선택
    • 배포 후 웹 앱 URL 복사
  3. 아래 폼에 설정 정보 입력
